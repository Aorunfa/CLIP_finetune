# CLIP_finetune
本项目在原CLIP仓库的基础上，新增损失、训练代码，主要用于对CLIP模型进行微调、完整理解CLIP算法的设计实现并从头训练。
模型参考于[CLIP](https://github.com/openai/CLIP)，损失参考于[open_clip](https://github.com/mlfoundations/open_clip)
